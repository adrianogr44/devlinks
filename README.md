🚀 DevLinks

Um projeto de página pessoal estilo Linktree, desenvolvido para centralizar meus links profissionais como LinkedIn, GitHub e outros, com suporte a modo claro e escuro e efeitos interativos com áudio.

✨ Funcionalidades
🌙 Alternância entre Modo Claro e Escuro
🔊 Efeito sonoro ao trocar de tema
🎨 Design moderno com Glassmorphism
📱 Layout Responsivo
🔗 Links para redes sociais
🚀 Animações suaves
🎯 Código simples e limpo
🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript
Google Fonts (Inter)
Ionicons
